# 小猫绑定 CatBinding

## 命令介绍

/xmcb add [item]: 无视nbt标签，将所有属于[item]的物品加入绑定清单
/xmcb remove [item]: 无视nbt标签，将所有属于[item]的物品从绑定清单中移除
/xmcb add-hand: 将主手中的物品（包含nbt标签）加入绑定清单
/xmcb remove-hand: 将主手中的物品（包含nbt标签）从绑定清单中移除
/xmcb reload: 重新加载配置文件
